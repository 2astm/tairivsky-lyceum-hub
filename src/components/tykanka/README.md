# Тиканка — React компонент

Клавіатурний тренажер для вивчення сліпого десятипальцевого набору. Написаний на React + TypeScript, готовий до вставки у будь-який Vite проект.

## Структура

```
src/components/tykanka/
  Tykanka.tsx        ← головний компонент (єдина точка входу)
  Keyboard.tsx       ← екранна клавіатура з кольорами пальців
  TaskDisplay.tsx    ← рядок тексту з курсором
  StatsBar.tsx       ← швидкість / точність / помилки
  CalibPanel.tsx     ← панель живого калібрування
  Modal.tsx          ← базова модалка
  HandsModal.tsx     ← SVG схема постановки рук
  hooks/
    useLayout.ts     ← визначення розкладки (API + калібрування)
    useTyping.ts     ← логіка набору, підрахунок балів, Telegram
  data/
    exercises.ts     ← 36 вправ + назви уроків
    layouts.ts       ← FALLBACK_LAYOUT, QWERTY_TO_UA, FINGER_MAP
  types.ts           ← спільні TypeScript типи
  tykanka.css        ← всі стилі з префіксом tyk-
  index.ts           ← barrel export
```

## Встановлення

1. Скопіюй папку `tykanka/` у `src/components/` свого проекту.
2. Ніяких додаткових залежностей — лише React 18+.

## Використання

```tsx
import { Tykanka } from './components/tykanka';

function App() {
  return <Tykanka />;
}
```

## Функціонал

- **36 вправ** — від окремих букв до повних текстів
- **Визначення розкладки** — KeyboardLayoutMap API (Chrome/Edge) з QWERTY→UA маппінгом; живе калібрування для Safari/Firefox
- **Живе калібрування** — друкуєш будь-які клавіші в довільному порядку, спочатку без Shift, потім з Shift; зберігається в `localStorage`
- **Кольори пальців** — кольорова смужка на кожній клавіші показує який палець відповідає
- **Домашній рядок** — крапка на клавішах А та О
- **Статистика** — симв/хв, точність, правильно/помилки в реальному часі
- **Оцінка** — від 0 до 12 балів (швидкість + точність)
- **Telegram** — результати відправляються вчителю після завершення вправи
- **Постановка рук** — SVG схема з підписами

## Налаштування Telegram

У файлі `hooks/useTyping.ts` замініть:

```ts
const TG_TOKEN   = 'ВАШ_ТОКЕН';
const TG_CHAT_ID = 'ВАШ_CHAT_ID';
```

## Стилі

Всі CSS класи мають префікс `tyk-` — не конфліктують з рештою проекту. CSS змінні визначені локально в `.tyk-root`, тому тема тренажера не залежить від теми хост-проекту.
